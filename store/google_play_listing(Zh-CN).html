LOCAL TRADER Local Trade 是一个LocalBitcoins网站的android开源应用。使用Local Trader你可以更新广告，管理交易，发送消息，释放和托管交易或者处理付款。使用Local Trader你可以通过LocalBitecoins的服务来处理Bitcoin，Ethereum，Dash，Ripple，Monero，Litecoin币种的在线或线下交易。 使用前你需要有一个Localbitecoins的账号，如果尚未注册，你可以在LocalBitcoins.com注册如果Localbitecoins在你的国家无法访问，你可以使用镜像网站LocalBitcoins.net注册. 请阅读入门指南 来学习如何在LocalBitecoins网站交易. 功能 ★ 支持Ethereum, Bitcoin, Dash, Ripple, Monero, Litecoin币交易 ★ 当你发送比特币时支持口令保护 ★ 释放或托管交易时需要口令保护 ★ 可以发送或接收关于交易的消息和通知 ★ 支持发送消息和下载消息附件 ★ 支持更新广告 ★ 支持搜索本地和线上的比特币买卖广告 安全 Local Trader 使用安全的Oauth2认证 来与LocalBitecoins服务沟通。 为了获得你得授权，你会被带到Localbitcoins的授权页面，你需要允许本应用访问权限，设置成功后，会返回到Local trader应用。 Local Trader需要Read、Write和Money_pin权限，当Local Trader替你发送、释放和托管比特币时需要Money_pin权限，你需要输入LocalBitcoins口令. 你需要在LocalBitcoins网站来设置你得口令，口令密码不是你得2FA（双因素验证）。 你可以在你的LocalBitecoins账号配置下查看和管理口令，对于被屏蔽的地区，你可以使用镜像网站. 权限说明 位置信息- 用来搜索和发布广告，只有在用户使用此功能时会使用你的位置信息； 同步设置- 用来后台同步数据和通知； 授权账号- 用来授权应用来同步数据。 其他信息 Local Trade不属于LocalBitecoins，如果遇到账号的相关问题请联系LocalBitecoins来获取支持。 → Localbitcions注册 → Localbitcoins支持 → Reddit社区 → Local Trader Github 项目 请在Local Trader Github 项目处提交程序bug或者功能需求。
