LOCAL TRADER Local Trader es una aplicación de código abierto de Android para LocalBitcoins. Use Local Trader para actualizar publicidades, gestionar intercambios, enviar y descargar archivos adjuntos de mensajes, liberar y financiar intercambios, contactar anunciantes o gestionar pagos. Con Local Trader puede intercambiar Bitcoin y Ethereum localmente o en línea utilizando el servicio LocalBitcoins.   Se requiere una cuenta LocalBitcoins para usar el operador Local. Para obtener una cuenta, regístrese en LocalBitcoins.com o use el nuevo sitio espejo en LocalBitcoins.net para regiones bloqueadas. Lea también las guías de inicio disponibles en LocalBitcoins.com para aprender a operar. CARACTERÍSTICAS ★ Comercio Ethereum y Bitcoin ★ Funcionalidad de billetera con código PIN para enviar Bitcoin ★ Liberar y financiar operaciones con código PIN ★ Enviar y recibir mensajes comerciales y notificaciones de actualizaciones de comercio ★ Enviar y descargar archivos adjuntos de mensajes comerciales ★ Cambiar la visibilidad del anuncio ★ Busque comercios locales o en línea para comprar o vender Bitcoin SEGURIDAD Local Trader utiliza autenticación Oauth2 segura para los servicios de LocalBitcoins.Para la autenticación, será llevado al sitio web de LocalBitcoins, donde deberá autenticarse como lo haría al iniciar sesión en LocalBitcoins. Tras la autenticación exitosa, será devuelto a la aplicación Local Trader. Permisos de solicitud de Local Trader para leer, escribir y money_pin durante la autenticación. El permiso money_pin le permite enviar dinero utilizando su código PIN de LocalBitcoins. Usted creó un código PIN como parte del registro de LocalBitcoins y es necesario para enviar Bitcoin de su billetera, y para financiar o liberar intercambios. El código PIN no es lo mismo que su Autenticación de 2 factores (2FA). Vea o administre su código PIN desde su página de perfil o use espejo de sitio para regiones bloqueadas. PERMISOS EXPLICADOS Ubicación: se usa para buscar anunciantes y crear anuncios nuevos, pero solo si el usuario lo solicita y otorga permiso a la aplicación. Escribir configuraciones de sincronización: se utiliza para sincronizar datos de fondo para notificaciones. Autenticar cuentas: se usa para la autoridad de la aplicación para sincronizar datos.
